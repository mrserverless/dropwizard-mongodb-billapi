[Dropwizard][1] prototype for a Bill retrieval API, written as part of a [Dropwizard Presentation][0]. Integrates 
with [MongoDB][2] via [MongoJack][3]. 
 
 Use gradle build and run:
 
     ./gradlew clean build run
